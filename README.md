# ESP32 Checkers AI (LovyanGFX + Joystick)

Полноценная реализация игры в шашки (Checkers / Draughts) для ESP32 с графическим интерфейсом на дисплее и управлением через аналоговый джойстик.

Проект включает:
- игровой движок с поддержкой обычных и дамок
- интеллектуального противника (AI) с алгоритмом поиска ходов (minimax / alpha-beta pruning)
- графический интерфейс с использованием LovyanGFX
- управление курсором через аналоговый джойстик
- систему выбора и подсветки доступных ходов
- обработку победы / поражения

Проект оптимизирован под embedded-устройства и может использоваться как база для собственных игровых разработок на ESP32.

Полноценная игра в шашки для ESP32 с графическим интерфейсом и встроенным искусственным интеллектом.

## 🧠 Как работает AI

ИИ использует алгоритм поиска (Minimax с ограничением глубины):

- Максимальная глубина задается:
  ```cpp
  #define AI_MAX_DEPTH 6
  ```

Чем больше глубина:

✔️ умнее игра
❌ медленнее ход

## 🔥 Особенности реализации

Все фигуры кодируются через enum:

enum Piece { EMPTY, WHITE, BLACK, WHITE_K, BLACK_K };

Ходы хранятся в массиве:

Move moves[64];

Минимизировано использование памяти

## 🧪 Как запустить

1. Установить Arduino IDE
2. Добавить поддержку ESP32
3. Установить библиотеку LovyanGFX
4. Подключить дисплей и джойстик
5. Залить Checkers2.ino на плату
